1. Quản lý nền Blog 2.0

          
Kết nối blog qua API (không lưu mật khẩu đăng nhập). Lưu lại để lần sau chỉ cần tick chọn. Tự đăng: WordPress (App Password), Dev.to, Hashnode. "Khác" = chỉ xuất bài để bạn dán tay.


          

            📖 Hướng dẫn lấy thông tin kết nối & tự đăng (bấm để xem chi tiết)
            

              
Chỉ các nền dưới đây tự đăng được (bấm Đăng là lên bài + trả link). Các nền không có API (Wix, Weebly, Site123, WordPress.com, Blogger qua app, exblog, pixnet, forumvi…) hãy chọn loại "Khác" rồi dùng nút 📋 Copy (giữ định dạng) để dán vào trình soạn blog.


              

                🟢 Dev.to — dễ nhất (khuyên dùng để thử)
                
    
                  
  1. Đăng nhập dev.to.
    2. 
                  
  2. Mở dev.to/settings/extensions.
    3. 
                  
  3. Tới mục "DEV Community API Keys" → nhập tên (vd SeoShark) → bấm Generate API Key.
    4. 
                  
  4. Copy chuỗi key vừa tạo.
    5. 
                  
  5. Tại đây: Loại nền = Dev.to → dán API key → đặt Tên gợi nhớ → + Lưu blog.
    6. 
                

              


              

                🔵 WordPress (TỰ HOST — có domain riêng)
                
    
                  
  1. Đăng nhập WordPress Admin: https://tenmiencuaban.com/wp-admin.
    2. 
                  
  2. Vào Users → Profile (Thành viên → Hồ sơ của bạn).
    3. 
                  
  3. Kéo xuống mục "Application Passwords" (Mật khẩu ứng dụng).
    4. 
                  
  4. Nhập tên (vd SeoShark) → bấm Add New Application Password.
    5. 
                  
  5. Copy chuỗi hiện ra (dạng abcd efgh ijkl mnop — GIỮ NGUYÊN cả dấu cách).
    6. 
                  
  6. Tại đây: Loại = WordPress → URL site (https://tenmiencuaban.com) + Username (tên đăng nhập WP) + dán App Password → + Lưu blog.
    7. 
                

                
⚠️ Cần WordPress 5.6+ và site chạy HTTPS. KHÔNG dùng được với wordpress.com (bản hosted) — chỉ dùng cho WP bạn tự cài trên hosting/domain riêng.

              


              

                🟣 Hashnode
                
    
                  
  1. Đăng nhập hashnode.com (tạo sẵn 1 blog/publication).
    2. 
                  
  2. Lấy token: mở hashnode.com/settings/developer → Generate New Token (Personal Access Token) → Copy.
    3. 
                  
  3. Lấy Publication ID: vào Dashboard blog của bạn → mục cài đặt publication; ID nằm trong đường dẫn dashboard (chuỗi chữ-số). Nếu không chắc, hỏi tôi gửi cách lấy nhanh.
    4. 
                  
  4. Tại đây: Loại = Hashnode → dán token + Publication ID → + Lưu blog.
    5. 
                

              


              

                ⚪ Telegra.ph (0 cấu hình — ở tab "Tự động Share Link")
                
Không cần kết nối gì. Vào tab 📣 Tự động Share Link → tạo nội dung → bấm 📝 Đăng Telegra.ph là tự tạo trang có nội dung + link (backlink) và trả về link ngay.

              


              
💡 Mẹo dán giữ định dạng: với nền loại "Khác", bấm 📋 Copy (giữ định dạng) rồi vào trình soạn bài của blog → bấm vào ô nội dung → Ctrl + V: tiêu đề, link anchor và ảnh sẽ giữ nguyên. Nhớ đặt bài ở chế độ Public khi đăng.
